Present Perfect Continuous Tenses
فعل حال مکمل جاری


Definition
In Present Perfect Continuous, the action has been taking place for some time and is still ongoing.
جو کام کچھ دیر کے لئے ہورہا ہو اور مسالسل جاری ہو فعل حال مکمل جاری کہلاتا ہے  



Affirmative Sentences
  Subject + has been / have been + (1st form of verb or base verb + ing) +
  object + time reference
مفعول + حوالہ وقت + ( ing + فعل کی پہلی فارم ) + has been / have been + فاعل  

  He has been watering the plants for two hours.
وہ دوگھنٹے سے پودوں کو پانی دیتا رہا ہے 

  I have been studying since 3 0’clock
میں 3 بجے سے پڑھتا رہا ہوں 



Negative Sentences
  Subject + has not been / have not been + (1st form of verb or base verb +
  ing) + object + time reference
مفعول + حوالہ وقت + ( ing + فعل کی پہلی فارم ) + has not been / have not been + فاعل  

  He has not been watering the plants for two hours.
وہ دوگھنٹے سے پودوں کو پانی نہیں دیتا رہا ہے 

  I have not been studying since 3 0’clock
میں 3 بجے سے نہیں پڑھتا رہا ہوں 



Interrogative Sentence
  Has / Have + Subject + been + (1st form of verb or base verb + ing) +
  object + time reference
مفعول + حوالہ وقت + ( ing + فعل کی پہلی فارم ) + has been / have been + کیا فاعل  

  Has he been watering the plants for two hours?
کیا وہ دوگھنٹے سے پودوں کو پانی دیتا رہا ہے 

  Have I been studying since 3 0’clock?
کیا میں 3 بجے سے پڑھتا رہا ہوں
